## Project Description:
    An application where the user can do the following tasks:
        - Create a to do task
        - Edit the task
        - View the Up Comming tasks
        - View all the listed task
        - Remove created tasks

## What was used:
    - JavaFX
    - XML
    - MVC
    - CRUD
